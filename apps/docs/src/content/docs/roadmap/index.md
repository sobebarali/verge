This section outlines planned features for Verge. These features are not yet implemented but represent the product vision.

## Planned Features

| Feature | Status | Description |
|---------|--------|-------------|
| [Web Scraping API](/roadmap/web-scraping-api/) | Planned | Core web scraping endpoints |
| [AI Extraction](/roadmap/ai-extraction/) | Planned | LLM-powered data extraction |
| [Google Sheets Integration](/roadmap/google-sheets-integration/) | Planned | Native Sheets integration |
| [Prompt Queries](/roadmap/prompt-queries/) | Planned | Natural language data queries |

## Vision

Verge aims to be the go-to AI data extraction tool for GTM teams, built directly into Google Sheets. The planned features focus on:

1. **Ease of Use** - Natural language prompts instead of complex selectors
2. **Reliability** - AI-powered extraction that adapts to page changes
3. **Integration** - Seamless Google Sheets experience
4. **Scalability** - Handle bulk scraping operations efficiently

## Contributing

When implementing roadmap features:

1. Review the feature specification in this documentation
2. Create a feature branch from `main`
3. Implement following existing patterns in the codebase
4. Update documentation as you build
5. Submit a PR for review
